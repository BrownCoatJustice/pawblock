PawBlock

Instructions

  • Use the status switch to turn PawBlock on and off. You can also quickly toggle it using the button in the browser toolbar.
  • Add rules to control which pages will be blocked by PawBlock.
  • The domain should be like "facebook.com", and the path should be like "/messages".
  • With this rule, "facebook.com/messages" will be blocked, but "facebook.com/profile" will be allowed.
  • To block a site entirely, just leave the path blank.
  • If something isn't working as expected, feel free to submit an issue or send me a message on Twitter.
